# ServerLog

A Vencord plugin that exports Discord server info into TXT, HTML, and JSON formats.

## Features

- HTML exports
- TXT exports
- JSON exports
- Export history
- Role export
- Channel export
- Emoji export
- Sticker export
- Cached member export
- Privacy mode
- Compact HTML mode
- Export presets
